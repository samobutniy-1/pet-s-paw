# Pet's Paw 🐾

A landing page for a pet adoption platform, built with clean HTML & CSS.

## About
Pet's Paw is a UI-only landing page designed to encourage people 
to adopt pets from shelters. The page presents the mission, 
organization info, and adoption process in a modern, visually 
appealing layout.

## Features
- Fully responsive layout built exclusively with CSS Grid — no Flexbox
- Hover animations on interactive elements
- Modern dark theme with purple accents
- Clean semantic HTML structure

## Note
This is a UI-only project — links and buttons are not functional.

## Tech Stack
HTML · CSS/SCSS (Grid only)
